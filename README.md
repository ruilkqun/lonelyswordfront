# lonelyswordfront
独孤九剑

## cnpm安装
```
npm install cnpm -g
```
## 一、安装webpack
```
root下：
npm install webpack webpack-cli  -g

sh-3.2# webpack -v
webpack 5.24.3
webpack-cli 4.5.0
```

## 二、安装vue-cli
```
root下：
npm install --global vue-cli

sh-3.2# vue -V
2.9.6
```

## 三、新建项目

vue init webpack change-your-bone

```

```
